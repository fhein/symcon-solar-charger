# symcon-solar-charger

IP-Symcon Modul zur solargestützten Steuerung einer WARP2-Ladestation auf Basis eines neutralen Energie-Gateways (z. B. Enphase).

- Library: `maxence` (library.json)
- Modul: `SolarCharger`
